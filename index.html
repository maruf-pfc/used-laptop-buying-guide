🔍 Used MacBook কেনার আগে যা Check করবেন

🛒 Pre-Purchase Checklist — MacBook
  1. Apple ID লক: চালু করে দেখুন — Activation Lock নেই তো? Setup করতে পারছেন? যদি seller-এর Apple ID লগইন থাকে সেটা sign out করাতে হবে।
  2. Serial Number verify: Settings > About > Serial Number → checkcoverage.apple.com-এ check করুন। Refurbished কিনা, কোন দেশের model তা জানা যাবে।
  3. Battery Cycle Count: About This Mac > System Report > Power → Cycle Count 500-এর নিচে ভালো, 800+ হলে battery change লাগতে পারে।
  4. Display দেখুন: Dead pixels আছে কিনা — সাদা/কালো full screen করে দেখুন। Backlight bleeding আছে কিনা।
  5. Keyboard & Trackpad: সব key press করুন। Trackpad click করুন সব জায়গায়।
  6. Ports সব test করুন: USB-C charger দিয়ে charge হচ্ছে? Thunderbolt port কাজ করছে?
  7. Memory & Storage চেক: About This Mac > Storage — advertised-এর সাথে মিলছে?
  8. কোন দেশের model: US/UK model best (Arabic keyboard হলে সমস্যা হয়)।
⚠️ বড় সতর্কবার্তা: Bangladesh-এ অনেক seller Intel MacBook-কে M1 বলে চালায়। About This Mac-এ গিয়ে "Apple M1" লেখা আছে কিনা নিজে দেখুন — processor "Intel Core" লেখা থাকলে সেটা পুরনো Intel model।
💡 Pro Tip: MacBook কেনার সময় macOS fresh install করে দিতে বলুন। তাহলে seller-এর সব personal data মুছে যাবে এবং আপনি নিজের Apple ID দিয়ে clean setup করতে পারবেন।

🔍 Used Windows Laptop কেনার আগে Check List

🛒 Pre-Purchase Checklist — Windows Laptop
  1. Generation confirm করুন: Settings > System > About-এ Processor দেখুন। 10th/11th/12th Gen Intel বা Ryzen 5000/6000 series কিনুন। Older-gen এড়িয়ে চলুন।
  2. RAM & Storage verify: Task Manager > Performance-এ RAM ও SSD দেখুন। HDD হলে SSD-তে upgrade করতে হবে।
  3. Battery health: PowerShell-এ "powercfg /batteryreport" run করুন। Design Capacity vs Current Capacity দেখুন। ৭০% এর নিচে হলে battery change দরকার।
  4. Keyboard ও Display: সব key press করুন। Screen-এ dead pixel, yellow spots দেখুন।
  5. Ports সব test করুন: USB, HDMI, headphone jack সব চেক করুন।
  6. Genuine Windows কিনা: Settings > Activation-এ "Windows is activated" লেখা আছে কিনা।
  7. Fan ও Cooling: YouTube 4K video চালিয়ে দেখুন — অস্বাভাবিক গরম বা আওয়াজ আছে কিনা।
  8. Original Charger: Original charger দিয়েই charge করুন — পরে third-party charger-এ সমস্যা হয়।
⚠️ ThinkPad কেনার বিশেষ সতর্কতা: Bangladesh-এ অনেক "refurbished" ThinkPad আসে Japan/Germany/USA থেকে। এগুলো generally ভালো, কিন্তু কিছু seller-রা worn-out battery-সহ বিক্রি করে এবং RAM বা SSD fake/lower-grade দিয়ে থাকে। কেনার আগে verify করুন।

Section 07

⚠️ Common Scams এবং কীভাবে বাঁচবেন

Bangladesh-এর used market-এ সবচেয়ে বেশি ঘটা ঠকানোর উপায়গুলো

🚨 Scam #1 — Intel কে M1 বলে চালানো: সবচেয়ে common। About This Mac-এ নিজে check করুন। "Apple M1" বা "Apple M2" না দেখলে কিনবেন না যদি M1 বলে দাম নেওয়া হয়।
🚨 Scam #2 — Lower RAM/Storage দেওয়া: "16GB RAM" বলে 8GB দেওয়া। About This Mac বা System Settings-এ verify করুন।
🚨 Scam #3 — iCloud/Activation Lock: কারো Apple ID লক করা MacBook কিনলে আপনি কখনো ব্যবহার করতে পারবেন না। দেখা মাত্র বাতিল করুন।
🚨 Scam #4 — Repaint করা বা Housing replace করা: পুরনো body-তে ভালো model-এর sticker/logo লাগানো। Original screws দেখুন — stripped হলে আগে খোলা হয়েছে।
🚨 Scam #5 — "Dead on arrival" বলে return না নেওয়া: কেনার আগে refund policy জেনে নিন। টাকা দেওয়ার আগে সব test করুন।
💡 Golden Rule: কোনো deal খুব ভালো মনে হলে সন্দেহ করুন। M1 MacBook যদি ৪০,০০০ টাকায় দেখান, নিশ্চিত থাকুন কোথাও সমস্যা আছে। Used market-এ fair price জেনে সেই অনুযায়ী কেনাকাটা করুন।
💡 Safe Buying Tip: সম্ভব হলে IT জানা কোনো বন্ধু বা পরিচিতকে সাথে নিয়ে যান। ৫০০–১০০০ টাকা দিয়ে হলেও একজন technician-কে দিয়ে check করিয়ে নিন — ৬০–৮০ হাজার টাকার বিনিয়োগে এটা ছোট খরচ।
